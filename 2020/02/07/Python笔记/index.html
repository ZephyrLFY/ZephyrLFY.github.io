四、函数

​ def function（）

  1. default argument：

    括号里可以指定默认值，也可以正常传入。

    在使用可选参数时，需要将其放到最后，否则会报错：non-default argument follows default argument

    Default argument的等号左右不要留空格

  2. 传参时是指针，不想被修改的话需要传入副本（列表需要用切片创造副本）

  3. 任意参数：def make_pizza(*toppings) 此处toppings是一个空元组，放置传入的所有参数

    在这个元组前可以加固定的参数

    还可以是空字典（键值对），例：**user_info
